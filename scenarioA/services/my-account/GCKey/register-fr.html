Modalités d'utilisation et Avis d’information

La CléGC est une identité électronique unique fournie par le gouvernement du Canada pour utiliser les services en ligne du gouvernement. Pour utiliser la CléGC afin d’accéder à Mon dossier Service Canada (MDSC), vous devez avoir :

  • un nom d’utilisateur CléGC;
  • un code d’accès.

Si vous avez présenté une demande de prestations d’assurance-emploi, du Régime de pensions du Canada (RPC) ou de la Sécurité de la vieillesse (SV) ou si vous recevez l’une ou l’autre de ces prestations, il se peut que vous ayez déjà reçu votre code d’accès. Vous avez besoin d'aide pour le trouver?

1. Demander un code d’accès

Le code d’accès est votre clé pour accéder aux services en ligne sécurisés. Avant de pouvoir vous inscrire à MDSC avec la CléGC, vous devez avoir un code d’accès.

Vous pouvez demander un code d’accès :

En ligne

Demander un code d’accès

Voici ce dont vous aurez besoin :

  • votre numéro d’assurance sociale (NAS);
  • le nom de famille de vos parents à la naissance;
  • votre code postal si vous êtes résident canadien, ou votre pays de résidence si vous êtes résident étranger.

Activer votre JavaScript.

En personne dans un bureau de Service Canada

Trouver un bureau de Service Canada

Voici ce dont vous aurez besoin :

  • votre numéro d’assurance sociale (NAS);
  • le nom de famille de vos parents à la naissance;
  • votre code postal si vous êtes résident canadien, ou votre pays de résidence si vous êtes résident étranger;
  • une pièce d’identité valide (non expirée) acceptable, comme :
    • un passeport canadien,
    • une certification de citoyenneté canadienne,
    • un certificat de statut indien,
    • un certificat d’enregistrement de naissance à l’étranger,
    • un permis de conduire (provincial ou territorial),
    • une carte santé,
    • une carte de résident permanent,
    • une carte d’identité provinciale.
Par téléphone

Présentez une demande en appelant Service Canada au 1‑800‑277‑9914 du lundi au vendredi, de 8 h 30 à 16 h 30 heure locale (vous pouvez aussi composer ce numéro pour vérifier l’état de votre demande de code d’accès personnel ou désactiver votre code d’accès personnel).

Voici ce dont vous aurez besoin :

  • votre numéro d’assurance sociale (NAS);
  • les réponses aux diverses questions de sécurité visant à vérifier votre identité.

2. Recevoir un code d’accès

Service Canada vous donnera un code d’accès. Vous devrez l’utiliser pour vous inscrire à MDSC.

En ligne :
Vous recevrez votre code d’accès par la poste dans un délai de cinq à dix jours ouvrables.
En personne :
Vous recevrez votre code d’accès immédiatement.
Par téléphone :
Le représentant de Service Canada du service téléphonique vous informera du moment où vous recevrez votre code d’accès.

3. Obtenir une CléGC (si vous n’en avez pas)

Utilisez votre CléGC pour accéder aux multiples Services en ligne du gouvernement du Canada.

Si vous n’en avez pas déjà une, inscrivez-vous au service CléGC en créant un nom d’utilisateur et un mot de passe. Si vous avez oublié votre nom d’utilisateur CléGC, vous devrez en créer un nouveau.

4. S’inscrire à MDSC avec votre CléGC

Vous êtes prêt à vous inscrire? Voici ce dont vous aurez besoin :

  • votre code d’accès;
  • votre numéro d’assurance sociale (NAS).
